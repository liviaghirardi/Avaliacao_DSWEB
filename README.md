# Avaliacao_DSWEB
# 💻 Avaliação DSWEB

Repositório destinado ao projeto de avaliação da disciplina de Desenvolvimento Web (DSWEB). O objetivo deste projeto foi desenvolver uma página web (focada em um currículo/portfólio) utilizando HTML e CSS, aplicando boas práticas de estruturação, estilização e organização de código.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Para a estruturação semântica e marcação do conteúdo.
* **CSS3:** Para a estilização visual, layout e tipografia.

## ✅ Critérios e Requisitos Atendidos

O projeto foi desenvolvido seguindo rigorosamente os critérios de avaliação propostos:

### 1. Estrutura e Organização do HTML
* **Documento Completo:** Estrutura base do HTML (`<html>`, `<head>`, `<body>`) implementada e funcionando corretamente.
* **Organização de Seções:** Uso de tags semânticas e `<div>`s para dividir a página de forma lógica.
* **Organização das Informações:** Separação clara e compreensível das seções de conteúdo, incluindo:
    * Dados pessoais
    * Experiência profissional
    * Formação acadêmica
    * Habilidades
    * Projetos e tecnologias

### 2. Estilização e Fidelidade Visual (CSS)
* **Fidelidade ao Modelo:** Layout geral estruturado para ser semelhante ao modelo proposto, respeitando posições e blocos principais.
* **CSS Básico Aplicado:** Uso adequado de cores, fontes e espaçamentos (margin/padding) para organizar a página visualmente.
* **Organização do Layout:** Elementos bem alinhados e distribuídos na tela.
* **Legibilidade:** Textos fáceis de ler, garantindo bom contraste, tamanho adequado de fonte e alinhamento correto.
* **Detalhamento:** Inclusão e estilização de elementos menores, como botões, textos auxiliares e divisões de seção.

### 3. Qualidade do Código
* **Código Organizado e Limpo:** Estrutura de fácil leitura e manutenção.
* **Indentação:** Correta indentação do HTML e do CSS.
* **Uso de Classes:** Nomenclatura coerente e padronizada de classes para a aplicação dos estilos.
